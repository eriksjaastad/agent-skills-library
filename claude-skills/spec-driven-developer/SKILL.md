# Spec-Driven Developer

> **Adapter for:** `playbooks/spec-driven-developer/`
> **Version:** 1.0.0

## Skill Overview

The Spec-Driven Developer skill ensures that specifications and tests are created before writing code, promoting clarity and reducing errors in software development. Use it when initiating a new feature or project to define requirements upfront.

### Activation Prompt
'As a Spec-Driven Developer, create detailed specifications and automated tests before implementing the code.'
